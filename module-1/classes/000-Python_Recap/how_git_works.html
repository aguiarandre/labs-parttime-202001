Git Setup

First things first.

git config

By the first time you use git on your computer, you should set your username and email as following:

git config --global user.name “FIRST_NAME LAST_NAME”
git config --global user.email "bla@example.com"

How to start working

Git works with the concept of Repository (Repo). You are able to have a local repo and remote repos. Let’s first introduce our local repository and see how it works.

git init

Initializes a .git folder in the folder you ran the command git init. It is the command that tells the software Git starts to track that folder.

git add

Until you say something, git doesn’t know which files you want to start versioning. You have to specify git add for git to start tracking that file.

git commit

Use git commit if you want to store the changes (4EVER).

git push

Git push is used to send all your modifications to your remote repository. Push will apply the modifications to all your files in the remote repo.

git pull

Git pull is used to bring all the updates from your remote repo to your local repo and merge those difference into your local files.

Basic Git workflow

In class you’ll mostly use

Branches

git branch new_branch

git checkout example_branch

Moves to the branch named example_branch

git checkout -b
