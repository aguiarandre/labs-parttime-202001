Main Terminal commands.

ls = list all files in the directory
ls -a = list all files in the directory including hidden files

pwd = print working directory
cd path/to/folder = change directory to path/to/folder
cd - = change directory to previous path
cd ·· = change directory to parent path
cd ~ or cd = change directory to home directory

mkdir path/to/new/folder = create a folder (make directory)
rmdir path/to/empty/folder = delete empty folder
rm -r path/to/non-empty/folder = delete folder recursively

man command or command --help = print at the output the manual of the command

cat path/to/file = print the content of the file at the output
echo “Hello” or echo Hello = print at the output the argument that you pass
> (example: echo Andre Aguiar > filename.txt) = redirect the output of a command to the file named filename.txt (by overwriting it)
>> (example: echo Andre Aguiar >> filename.txt) = redirect the output of a command to the file named filename.txt (by appending it)
